Initial stable release of aura-ai-chat! A production-grade, framework-agnostic AI chat widget built as a Web Component. Features include an agentic loop with iteration tracking, Human-in-the-loop tool approvals (safe, moderate, destructive), WebMCP integration, a built-in GitHub Copilot provider, and extensible interfaces for custom providers and conversation persistence.
